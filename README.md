## 🛠️ Technologies: 
- Terminal: **Alacritty** for it's GPU rendering (Because I was upset at Iterm, and it's simpler).
- Shell: **Bash** zsh is ugly, jk I just think bashrc is easier to configure, but I have never looked at the zshrc so who really knows.
- Text Editor: **NeoVim** I literally made it look as much like vscode as possible, but I never have to use my mouse, so thats the real plus.
- Window Manager(?): **tmux** Beutiful, also made it look like vscode, but it helps be not use MacOS's window manager lol.
- Browser: Arc (Unless I am doing literally anything important, which requires a chrome extension haha)
